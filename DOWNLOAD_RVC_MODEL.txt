ISSUE 1: "RVC Model not found"
-------------------------------
Solution:
- Check file name: must be EXACTLY "vietnamese_female.pth"
- Check location: must be in "models/rvc/" folder
- Check path: use absolute path to verify


ISSUE 2: Model file too small (< 20MB)
---------------------------------------
Solution:
- File is corrupted or incomplete
- Re-download from different source
- Try OPTION C (Google Drive backup)


ISSUE 3: Import error / module not found
-----------------------------------------
Solution:
- Install dependencies: pip install -r requirements.txt
- Check rvc_engine.py exists
- Restart Python/server


ISSUE 4: Hugging Face upload fails
-----------------------------------
Solution:
- File too large for git (> 100MB)
- Use git-lfs: git lfs install
- Or upload via web interface


ISSUE 5: Still sounds like reading voice
-----------------------------------------
Solution:
- Model might not be a singing model
- Try different model from voice-models.com
- Look for "singing" or "vocal" in model name
